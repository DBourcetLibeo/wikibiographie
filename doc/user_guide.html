Vider les biographies Wikipdia en cache

Afin d'viter de multiplier les appels Wikipdia chaque consultation d'une biographie, l'extension WikiBiographie utilise la fonctionnalit de "transient" de WordPress. Cela signifie que les informations biographiques provenant de Wikipdia sont stockes pour une certaine dure, on dit qu'elles sont mises en cache. Les informations sont de nouveau rcupres de Wikipdia lorsqu'une biographie est consulte sur le site et que les donnes mises en cache sont expires.

S'il est ncessaire de supprimer toutes les informations mises en cache, il suffit de cliquer sur le bouton "Vider les donnes mises en cache".

Ce paramtre permet de limiter le nombre caractres affichs de l'information "Description" sur la page d'une biographie (ne concerne pas "Description complmentaire" qui est un champ libre). Mettre la valeur "0" (zro) pour ne pas limiter le nombre de caractres affichs.
